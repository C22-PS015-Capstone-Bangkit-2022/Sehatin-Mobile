# SEHATIN

An Application that can provide to user the recommended food and exercises based on user's diseases

### Features
- Login
- Register
- Edit Profile
- Recommended Food
- Recommended Exercises
- Food's Nutrition Detection
- Social Media (Post & Chat)
- Consultation
